## 0. О ЧЁМ ИГРА

**В двух словах (для друзей):** несёшься через неоновый киберпанк-город на бешеной скорости — бежишь по стенам, прыгаешь между крышами, скользишь под препятствиями. Чем чище проходишь — тем быстрее летишь. Один забег = несколько секунд чистого потока и адреналина. Потом сравниваешь время с друзьями и всем миром: рядом бежит призрак чемпиона, и ты пытаешься его обогнать. Жанр одной строкой: **скоростной паркур-рейсер от 3 лица — будто Ghostrunner встретил Trackmania.** Игра про скорость, точность и желание пройти трассу ещё чище.

Скоростная паркур-гонка от 3 лица: максимальная скорость, цена ошибки в долю секунды, накопление импульса за чистоту прохождения. Соревнование через призраков и лидерборды на сложных трассах — авторских и генерируемых. Киберпанк.
Референсы: Ghostrunner (поток), Warframe (импульс), Trackmania (соревнование/дейлики), Neon White (скорость+лидерборды), Titanfall 2 (фил).

## 1. ПОРЯДОК РАБОТЫ (железно)
80% успеха = game feel. Проверяется первым, на серой болванке, ДО всего.
1. Фил — манекен на gym-полигоне (см. §1-бис) + камера.
2. Одна ручная трасса + таймер + рестарт + свой призрак. Дать друзьям.
3. Призраки + лидерборд (асинхрон).
4. Правила трасс → модульная библиотека → дейлики (генерация).
5. Тюнинг статов (опц., отдельный режим).
6. Realtime-сеть — в самом конце, если вообще нужна.
Каждый этап дешевле провалить, чем следующий.

## 1-БИС. GYM-ПОЛИГОН ДЛЯ ФИЛА (18.06.2026)
Этап 1 — НЕ пустая коробка, а минимальный серый полигон: по одному эталонному элементу на каждую механику мувсета. Иначе нечего тюнить (вал-ран без стены не настроить, слайд без горки — тоже).
Элементы (серые блоки, без графики):
- **Пропасть** (провал) → дальность прыжка, рывок, coyote time, jump buffer.
- **Стена сбоку** → вал-ран (горизонтальный бег по стене).
- **Вертикальная стена / уступ** → вскарабкивание (climb / mantle через край).
- **Горка-рампа** → слайд-скат + набор импульса вниз.
- **Серия платформ** → чейнинг прыжков, сохранение импульса между приёмами.
Эти эталонные элементы — первые кирпичи модульной библиотеки §6 (gym перерастает в стройматериал, не выкидывается).
**Мувсет (дополнен):** бег, прыжок, рывок (КД), вал-ран, слайд, **вскарабкивание/мантл**.
**Рывок — по КД (решено), и это намеренно тактический ресурс.** КД делает ценным не сам рывок, а ТАЙМИНГ его траты. Следствия (источник глубины маршрутов без ручного дизайна):
- Тайминг важнее дистанции: путь чуть длиннее выгоден, если ставит рывок в идеальную точку.
- Комбо-ценность: рывок в слайд / рывок с горки даёт куда больше скорости, чем рывок «в пустоту» → линия строится вокруг таких связок.
- Цена ошибки: растратил рывок не вовремя → ждёшь КД, теряешь темп. Прямой путь с пустым рывком проигрывает кривому с эффективным.
Так развилки риск/награда (§5) рождаются сами, из взаимодействия КД и геометрии — игрок считает тайминги.

## 1-ТЕР. ПЕРВЫЙ ПЛЕЙБЛ — вертикальный срез для друзей (решено 18.06.2026)
Минимум, чтобы фил читался, и ни байта больше. Цель — честный ответ «тянет ли перепройти». Это шаг 2 из §1.
Что входит:
- **Одна трасса** из gym-элементов (§1-бис): пропасть → вал-ран → мантл → спуск-слайд → финиш. Серые блоки, без графики.
- **Спуск — кульминация в конце.** Раз спуск наш фокус, трасса ведёт к эпичному скоростному спуску, чтобы забег заканчивался на пике ощущения.
- **Таймер + мгновенный рестарт** (одна кнопка, без меню — критично для «ещё разок»).
- **Свой призрак — со 2-го заезда, ТОЛЬКО лучший (PB).** Один призрак = личный рекорд, перезаписывается при побитии. Не каждый заезд и не пачка — один эталон, твой потолок. Первый заезд его создаёт.
- **Спидометр** (приборка §3.3-кватер) — виден разгон.
Что НЕ входит: лидерборд, сеть, графика, вторая трасса, звук кроме ветра.
Суть: маленький срез, но содержит всю петлю «беги → ошибись/спасись → финишируй быстрее → ещё разок». Друг за 5 минут понимает игру.

## 2. КАМЕРА — 3 ЛИЦО (решено)
Видно персонажа (косметика + читаемость трассы), лучше пространственная осведомлённость, драйв через тело. Прецеденты: Sonic, Bomb Rush Cyberfunk, Solar Ash, Severed Steel, Pseudoregalia.
Камера = половина фила: слежение на скорости/поворотах, поведение при падении, не врезаться в стены, FOV-буст + наклон.
⚠️ На болванке попробовать обе камеры, решить руками.

## 3. GAME FEEL — ОБЯЗАТЕЛЬНОЕ
Суть: расширять окна по времени/позиции, чтобы игре подыгрывать игроку.
3.1 Базовый фил:
- Coyote time — прыжок ещё ~5 кадров после схода с края.
- Jump buffering — нажатие чуть раньше земли срабатывает в кадр касания.
- Corner correction — задел угол → подталкивает мимо, не клинит.
- Halved-gravity jump peak — на вершине прыжка гравитация вдвое слабее.
3.2 Камера/обратная связь (продают скорость):
- FOV растёт со скоростью (главный трюк).
- Наклон камеры на вал-ране/поворотах.
- Поджатие ног / руки в кадре на рывке/слайде.
- Звук + лёгкий motion blur на пике.
- Микро-тряска на приземлении/рывке.
3.3 Ядро — накопление скорости:
- Momentum preservation — импульс не сбрасывается между движениями.
- Накопление за чистоту — дольше без столкновений → быстрее. Видимый индикатор.
- Сброс при ошибке — врезался = потерял разгон.
- Momentum storage — прыжок с движущейся платформы добавляет её импульс.
- Мувсет: рывок (КД), вал-ран, слайд с поджатием, прыжок, вскарабкивание/мантл.

3.3-бис ЗАКОН ВЫСОТЫ = СКОРОСТИ (решено 18.06.2026): сохранение энергии, гравитация — друг на спуске.
- **Вниз → разгон.** Слайд по склону вниз копит скорость. Вал-ран по НИСХОДЯЩЕЙ стене — разгоняет (не просто держит).
- **Горизонт → сохранение.** Ровный вал-ран / бег держит импульс, не растит.
- **Вверх → плата.** Вскарабкивание, подъём, восходящий вал-ран — тратят накопленное.
Следствие для левелдизайна: трасса = управление высотой. Забрался наверх (дорого, медленно) → конвертируешь в скорость на спуске. Цикл «заберись — слети». Высота = запасённая скорость; спидранер думает «где набрать высоту, чтобы слететь быстрее», не «где короче». Рефы: SSX, Tony Hawk, Dying Light, реальный паркур.
⚠️ Гравитация-через-импульс FPS-зависима (RESEARCH §2 — главный риск честности) → спуски разгоняли бы по-разному на разном железе. Лечится ФИКС-ТАЙМСТЕПОМ (уже приоритет №1). Цена фичи = дисциплина по детерминизму.

3.3-тер СПАСЕНИЕ ПО ТАЙМИНГУ — wall-tech / kick-off (решено 18.06.2026). Столкновение НЕ бинарный провал, а лестница опций под уровень игрока (парирование как в Sekiro, для геометрии; рефы Titanfall, Mirror's Edge). 4 исхода от safe к pro:
1. **Объезд** — не берёшь рисковую линию. Медленнее, но цел. Доступно всем.
2. **Блок (ЗАЖАЛ кнопку)** — теряешь ЧАСТЬ импульса, но не всё. Тайминг не нужен, просто среагировал. Подушка для новичка.
3. **Парирование (попал в УЗКОЕ окно)** — сохранил импульс ПОЛНОСТЬЮ, лучше — перенаправил (отскок вперёд/вверх) + лёгкий буст. Про-игрок СПЕЦИАЛЬНО режет углы вплотную ради ускорения.
4. **Ничего / лоб на полной** → полный сброс импульса + микростан. Цена настоящей ошибки.
Правила: (а) АНТИСПАМ — окно парирования узкое, привязано к контакту; зажать заранее = только блок (п.2), не парир. (б) ЧЕСТНОСТЬ — окно одинаково на любом FPS → опять ФИКС-ТАЙМСТЕП.
Порог входа (РЕШЕНО — Trackmania-модель): темпом НЕ давим, безопасный путь всегда есть. Скорость нужна ради ТАБЛИЦЫ, не ради «пройти вообще». Один контакт читается 4 способами под 4 уровня игрока — никого не выгоняем (скилл-срез §5-бис). Отскок вверх → высота → скорость (кормит 3.3-бис).
3.3-кватер ОБРАТНАЯ СВЯЗЬ СКОРОСТИ — спидометр + сенсорика (решено 18.06.2026). Два РАЗНЫХ слоя, нужны оба, не подменять:
- **Спидометр = информация (мозг).** Для спидран-игры must-have: точное число → оптимизация линии, «здесь потерял 3 м/с». Рефы: Trackmania, Distance, bhop-ups.
- **Сенсорика = ощущение (тело).** FOV-кик + **ветер** (свист: громкость и тон растут со скоростью — главный дешёвый аудио-сигнал разгона; визуально — стрик-линии/частицы мимо камеры, искажение по краям экрана) + тряска + motion blur на пике. Без неё спуск вял даже при растущей цифре.
- ⚠️ Ловушка: только спидометр → игрок втыкает в цифру, не в мир, скорость не ЧУВСТВУЕТСЯ. Спидометр ДОПОЛНЯЕТ сенсорику, не заменяет.
- **Приборка спидранера:** спидометр + индикатор импульса/чистоты + таймер + сплиты. ОПЦИОНАЛЬНА/настраиваема (трейлер и новичок — спрятать, чистый кадр à la Mirror's Edge; оптимизатор — развернуть). Показывает горизонтальную скорость (вертикаль — решить на болванке).
- **Бонус:** спидометр обучает невидимым системам — видно, как цифра растёт на спуске / падает на подъёме (3.3-бис), как импульс капает за чистоту. Делает наши законы считываемыми + доверие к системе.

3.4 Философия: скорость ЗАРАБОТАНА, не выдана (Tribes, Quake, Titanfall, Ultrakill). Neon White: «игра должна научить тебя быть быстрым».

3.4-бис МУЗЫКА И ЗВУК (решено 18.06.2026).
- **Тон: жёсткая электроника — darksynth / drum‑n‑bass / industrial.** Агрессивно, качает, киберпанк. Школа Ghostrunner (Daniel Deluxe, Dan Terminus, GosT). НЕ эмбиент Mirror's Edge — у нас движуха.
- **ADAPTIVE, не ритм‑игра:** музыка РЕАГИРУЕТ на игрока, не наоборот. Слои (stems) нарастают с импульсом/скоростью — на чистом разгоне трек расцветает, на смерти/ошибке проседает (теряет слой). Вертикальное наслоение (FMOD/Wwise). Ритм‑игру (попадать в бит, Hi‑Fi Rush) НЕ берём — убьёт свободу линий и честность.
- **Музыка = второй индикатор импульса** (ушами): расцвела = в потоке; оборвалась = облажался. Аудио‑награда за чистоту, дублирует ядро 3.3. На эпичном спуске трек на полную — продаёт кульминацию.
- 🚨 **Музыка клиентская, на детерминизм НЕ влияет** (поверх симуляции, не трогает физику/тайминги).
- 🚨 **Болванку тестим БЕЗ музыки** (тишина + ветер + шаги). Музыка маскирует кривой фил → кажется кайфово из‑за трека, а движение сырое. Фил вкусный сам по себе в тишине, музыка ПОСЛЕ. Правило.
- **Продакшн:** Рамиль спросит знакомого музыканта (composer‑партнёр). На болванке музыки нет; темп‑треки только для прототипа трасс. Не бюджетный блокер.

3.5 НЕ делать на болванке: графику, трассы, Lumen/Nanite, бой, меню, тюнинг, модель, МУЗЫКУ (фил — в тишине).

## 4. ФИЗИКА И АРХИТЕКТУРА
Физику не планируют — настраивают итерациями. Фил живёт в пальцах. Планируешь возможность БЫСТРО крутить:
- Все параметры наружу (рывок/откат/гравитация/эйр-контроль) — менять в редакторе на лету (BP/DataAsset).
- Числа независимы.
- Быстрый цикл правка→проверка — главный фактор скорости.
- ⚠️ Не делать «идеальную песочницу» месяц. Достаточно удобную за пару дней.
4.1 КРИТИЧНО: фиксированный таймстеп физики (не к фрейму). Иначе на разном FPS разная скорость → лидерборды нечестные. UE: DeltaTime + фиксированный тик. Заложить сразу.

## 5. ЛЕВЕЛДИЗАЙН ТРАСС

**РЕШЕНО (18.06.2026): ЧАСТЬ 1 = ЗАКРЫТЫЕ ТРАССЫ-УРОВНИ, НЕ открытый мир.**
Почему уровни: (1) можно выкатывать контент частями, по карте за раз; (2) каждую карту дорабатываешь персонально, доводишь до кайфа; (3) на закрытых трассах отрабатываются основы — все великие гонки начинали так. Цель части 1 — пара трасс, по которым реально интересно бегать.
**Путь роста контента — лестница, не прыжок: ТРАССА → РАЙОН → ГОРОД.**
Когда накопится несколько трасс в одном сеттинге — сшить их в РАЙОН: общее бегабельное пространство, старые трассы = маршруты внутри него + связки между ними. Это мини-открытый мир контролируемого размера. Районы стыкуются → постепенно вырастает город. Каждый район — отдельная порция контента (катим частями), риск нарастает плавно, переиспользуются те же модули §6. Город = много сшитых районов.
**Открытый город — видение ЧАСТИ 2** (если доживём): трасса = A→Б по якорям внутри города, поиск маршрута на игроках. Модель — Mirror's Edge Catalyst. Технически реально на World Partition (UE 5.8), переиспользует модульную библиотеку §6. Сейчас НЕ строим — сложность мира убивает отработку основ.

5.1 «Широкий коридор»: явное направление А→Б, внутри линии риск/награда:
- Безопасная — длиннее, прощает, для новичка.
- Быстрая — короче, нужен точный вал-ран/рывок/импульс.
- Безумная — для задротов, 1 из 100, потолок скилла.
Ключ: линии дают РАЗНОЕ время. Скилл = выбор линии + чистота.
5.2 НЕ полная песочница — свобода ломает «правильную линию» → лидерборд разваливается.
5.3 СМЕРТЬ И ФЕЙЛ-СТЕЙТЫ (решено 18.06.2026). Без смерти у риска нет цены: ошибка = только потеря скорости → рисковая линия «бесплатна». Смерть-зоны дают риску настоящие зубы («цена ошибки в долю секунды»). Два типа фейла:
- **Мягкий — потеря импульса.** Лоб в стену / промах wall-tech (§3.3-тер) → сброс скорости + микростан. Не смерть, бежишь дальше медленно.
- **Жёсткий — смерть.** Улетел в пропасть / смертельную зону (киберпанк: обрыв, лазеры, провал) → МГНОВЕННЫЙ рестарт.
Правила: (а) смерть ДЕШЁВАЯ и мгновенная — рестарт по той же кнопке, без загрузок и наказания (цикл «ещё разок» священен; Neon White, Trackmania). (б) смерть-зоны — атрибут РИСКОВЫХ линий, НЕ основного пути: безопасная линия (Trackmania-порог входа §3.3-тер) без пропастей под ногами, медленно но не убивает. Хочешь срезать над бездной по безумной линии — там и ставка: влетел красиво или сдох.

## 6. DATA-DRIVEN ПАЙПЛАЙН ТРАСС
1. Фил → цифры (дальность рывка, высота прыжка, скорость вал-рана, импульс).
2. Правила/размеры → из цифр (мин/макс разрыв, длина стены, высота). «Словарь» левелдизайна.
3. Пресеты комбо-трюков → вылизанные модули, гарантированно проходимы.
4. Динамическая генерация → из проверенных модулей (WFC, не рандом).
5. Текстуры → последний слой.
6.1 Шаг 2 — это КОД: правила в данные = единый источник истины (ручная сборка + генератор + валидация). Поменял рывок → всё пересчиталось.
6.2 ⚠️ Генератор не раньше времени: сначала 10–15 модулей руками на реальном филе.
6.3 Процедурка vs честность: сид-генерация (у всех одна daily-трасса) / модульная сборка / процедурка как инструмент дизайнера.

## 7. ДВА РЕЖИМА КОНТЕНТА
- Curated (авторские, убер-сложные) — мастерство, киберспорт, стримеры. Ядро/лицо.
- Динамические (сид/daily) — свежесть, удержание, равные условия. Ретеншн-движок.
Один фундамент: модульная библиотека кормит оба.

## 8. СОРЕВНОВАНИЕ — АСИНХРОН, НЕ REALTIME
Призраки + лидерборды + дейлик по сиду.
- Все успешные (Trackmania, Neon White, Distance) — через призраков.
- Realtime-неткод на скорости = смерть для малых команд (Lemnis Gate закрылся, Distance отменил онлайн).
- Призраки = ~90% драйва за ~10% сложности, всегда доступны.
Опц. «live» = призраки-оверлей без коллизии.

## 9. ТЮНИНГ СТАТОВ (Forza-style, опц., рискованный)
⚠️ **ПЕРЕСМОТРЕНО 19.06.2026 (см. §5т.10):** введена ВЕРТИКАЛЬНАЯ прокачка (Рамиль решил). §9 ниже («НЕ прокачка, бюджет равный») больше НЕ базовая модель — теперь это ЭНДГЕЙМ-слой: после soft-cap прокачки вертикаль переходит в билды-сайдгрейды (этот §9). До капа герой растёт вертикально.
НЕ прокачка, а тюнинг: общий бюджет одинаков, перераспределяешь внутри. Один робот, разные сетапы. Честно.
- Модули = сайдгрейды (меняют КАК двигаешься, не «быстрее»).
- Раздельные лидерборды: Spec (равны, киберспорт) + Open.
- Косметика без потолка, статы — нет.
- Потолок скорости от исполнения, не от модуля.
⚠️ В FPS-мувменте тюнинга почти нет. Вводить поздно, за флагом. Kill-switch: solved meta за недели → увести в casual.

## 11. ПРОИЗВОДИТЕЛЬНОСТЬ (FPS на длинных трассах)
Высокий FPS — требование жанра. Минимализм = и эстетика, и оптимизация. Не гнаться за фотореализмом (на скорости в смаз + вредит читаемости).
- World Partition — авто-стриминг, длина трассы почти не влияет на FPS. Уже в UE5.
- Occlusion culling — повороты/стены перекрывают обзор.
- Distance culling / HLOD.
- Модульность — дешёвый инстансинг.
- На болванке выключить Lumen/Nanite (300+ FPS).
⚠️ На скорости стриминг должен ОПЕРЕЖАТЬ игрока: радиус предзагрузки с запасом + async. Тестировать на макс. «вжухе».

14.2-тер **ПРОТАГОНИСТ + МОДЕЛЬ КАСТОМИЗАЦИИ (решено 20.06.2026) — личность задана, тело свободно.** Развязка конфликта «месть требует заданного героя» vs «хотим любой облик (М/Ж/андрогин/монстр/что угодно)»: герой — КИБОРГ, **личность ≠ тело**. Постоянна ЛИЧНОСТЬ (голос, почерк, история, прозвище), тело — СМЕННЫЙ КОРПУС.
- **Подстава = обнулили кибернетику (ядро находки).** Был легендарным ассасином с топовым железом → подстава (§кампания) = аугментации выжгли/занесли в чёрный список/содрали → очнулся в дешёвом утильном корпусе на дне. Один поворот объясняет ВСЮ механику: **прокачка = заново вживляешь/апгрейдишь кибернетику** (гринд обоснован лором); **🔵 голый класс §5т.10 = базовое шасси / 🟣 прокач = аугментированный** (два лидерборда на лоре); перки-двери (§358 — пробитие стекла/стен, крюк) = новые импланты.
- **«КИБОРГ» = ВНУТРЕННЯЯ УСЛОВНОСТЬ, НЕ визуальная обязаловка (уточн. 20.06.2026).** Кибернетика сидит ВНУТРИ и лорно объясняет, ПОЧЕМУ умеешь бегать по стенам / видеть конусы / двигаться быстрее. Следствие — чистое разделение: **облик = косметика (0 влияния на геймплей), прокачка = внутренняя кибернетика (невидимая/опц.-видимая)** → облик ПОЛНОСТЬЮ развязан с прогрессией (не p2w по виду; «обнулили железо» меняет СПОСОБНОСТИ, не внешку).
- **АРТ-НАПРАВЛЕНИЕ ОБЛИКА = STELLAR BLADE (решено 20.06.2026), НЕ хром-протезы CP2077.** Персонаж — СТИЛИЗОВАННО-КРАСИВЫЙ ЧЕЛОВЕК (М/Ж) в облегающем костюме/техвире; «киборг» = ИМПЛАНТЫ НАМЁКОМ (под кожей / лёгкие техно-акценты), НЕ замена конечностей, НЕ торчащее железо. Чисто, гладко, секси-sci-fi. Разводит референсы: **Cyberpunk 2077 — только город/неон/вертикаль (§14.1-бис, фон), а ТЕЛО = Stellar Blade.** Сходится с секс-эппилом (§14.2) и косметикой-монетизацией (костюмы = главный товар). Хром/монстр — опц. крайность косметики на потом, НЕ ядро вайба.

Решить ПОЗЖЕ (после фила, арт — последний слой §14.4).

- **ОТКРЫТО:** (1) ПРОЗВИЩЕ (варианты: НОЛЬ — бьёт в обнуление, нейтрален к фракциям, ставка; РОНИН — «без господина»=фрилансер, но самурайский крен к якудзе; БЛИК; ВЕКТОР); (2) КЕМ БЫЛ ДО ПОДСТАВЫ — (а) лучший киллер одной из фракций (предала своя, личная драма, но привязка к фракции) vs (б) независимый «номер один» города (подстава=сговор, надфракционный, чище ложится на «качай всех» §391) → тянет за собой АНТАГОНИСТА.

## 5-БИС. ЛЕВЕЛДИЗАЙН — ПЕРЕОСМЫСЛЕНО (18.06.2026): ЧЕКПОИНТЫ-ЯКОРЯ + СВОБОДА МАРШРУТА
Корректирует §5. Игра = площадка для спидранов → emergent-маршруты это ФИЧА, не баг.
5б.1 **Два вида срезок (различать жёстко):**
- ЧИТ-срез = эксплойт коллизии / физ-баг / телепорт сквозь геометрию → УБИРАЕМ.
- СКИЛЛ-срез = через механики (накопил импульс, тормознул ради разгона, точный безумный прыжок, хитрый маршрут) → **СВЯТОЕ, не трогаем.** Душа спидрана.
5б.2 **Чекпоинты = ЯКОРЯ, не рельсы:** говорят «здесь надо реально побывать на скорости» (против читов мимо трассы), но НЕ говорят КАК их собрать. Между якорями — полная свобода маршрута. Игрок сам планирует: где тормознуть для разгона, откуда прыгнуть, как срезать импульсом.
5б.3 **3 линии (безопасная/быстрая/безумная) — ПОДСКАЗКИ-ориентиры, не рельсы.** Мастер найдёт 4-ю линию, которую не задумывали — это победа дизайна, не провал. Skill ceiling бесконечный.
5б.4 **Философия «не патчить emergent»** (Quake/Titanfall/Ghostrunner): bunny hop, rocket jump родились багами → стали ядром жанров. Патчить ТОЛЬКО то, что ломает физику насквозь; неожиданные маршруты через механики — оставлять.
5б.5 **Искусство дизайна якорей:** ставить чекпоинты так, чтобы их нельзя собрать «не по духу» (через дыру коллизии), но маршрут между ними максимально свободен. Узкие якоря, широкая свобода.
5б.6 **Баланс линий доказывать ЛОГАМИ** (время/линия, кто что выбирает), не интуицией — тот же логгер, что в тюнинг-стенде §22. Читаемость развилок на скорости (визуальный язык: цвет/форма/свет). Чекпоинт = 3 функции: валидация заезда + splits таймера + точка рестарта practice (§23).

## 5-ТЕР. ГЕЙМПЛЕЙ-ЦЕЛЬ: КОНТРАКТЫ — ЗАКАЗНЫЕ УБИЙСТВА (решено 19.06.2026)
Чистый «доехал быстрее» приедается → нужна ЦЕЛЬ и ДАВЛЕНИЕ сверх бега. Решение: **кампания контрактов** — ты гиперскоростной ассасин в киберпанк-мегаполисе. Фантазия: **скорость = стелс** («призрак, которого видят только мёртвым» — пронёсся так быстро, что засекли лишь те, кого уже убил). Триангуляция, у которой нет прямого конкурента: **раутинг Хитмана × движение Mirror's Edge/Ghostrunner × зрелищность AC Contracts**, в киберпанке.
🚨 **Это НАДСТРОЙКА цели поверх движения, НЕ замена спидран-ядра.** Детерминизм (§23, приоритет №1), призраки (§8), тайминги — всё сохраняется; контракты дают им СМЫСЛ. НЕ для болванки и не для первого плейбла (§1-тер): сначала фил/физика кайфовые, контракты — слой СОДЕРЖАНИЯ части 1, появляется после.

5т.1 **Дыхание уровня (три такта):** снаружи РАЗГОН (открытый паркур по крышам/фасадам, копишь высоту=скорость §3.3-бис) → внутрь УДАР (влетел в окно → толпа/вечеринка → на скорости режешь цель, не останавливаясь) → наружу ВЫБРОС (вылетел в след. окно, скорость сохранил, флоу дальше). Влетел-резанул-вылетел = 2-3 сек, пик-момент комбо, НЕ комната-лабиринт. Здание прошивается насквозь.

5т.2 **Мир детерминированный, как часы (модель Хитмана) — фундамент спидрана.** Спидран Хитмана живёт только потому, что цели/NPC ходят по ЖЁСТКОМУ расписанию (позиции, тайминги заучиваемы). Нельзя оптимизировать рандом. Значит цель — НЕ импровизирующий AI, а **clockwork**: в секунду X всегда в точке Y. Игрок учит распорядок → строит нереальные тайминги-маршруты от позиций цели (ровно то, на чём держатся спидраны Хитмана). Согласуется с детерминизм-приоритетом №1 (§23).

5т.3 **Цель живая, ФАЗЫ по времени — твоя скорость решает ГДЕ застанешь (давление):**
- **0–15с — балкон, одна, курит.** Идеальный килл, ранг S, окно узкое.
- **15–35с — ушла в зал на тусу.** В толпе: прошить сквозь людей, опознать на скорости, средне.
- **35с+ — охраняемая комната / лифт / уезжает.** Хард: погоня, прорыв или вообще упустил.
Геометрия миссии зависит от тебя. Был быстр — взял на балконе. Затупил — цель сместилась, **перестраиваешь дугу на лету** (reroute) к другому окну. Это «скорректировать маршрут на скорости» = ядро скилла; живая цель ЛОМАЕТ зубрёжку чистого спид-рана (каждый забег чуть иначе — читаешь, куда ушла). Таймер перестаёт быть абстрактным — воплощён в человеке, которого теряешь по этажам.

5т.4 **Трекер цели — честность.** Пульсирующая точка/силуэт сквозь стены (та же wall-tech-подсветка) → на скорости видно, куда сместилась, не угадываешь вслепую. Толпа-статисты = декорация флоу; цель = единственный «горячий» силуэт. Опознать надо ДО входа (ещё с крыши), чтобы не тормозить разглядывать.

5т.5 **Выбор стиля маршрута (осознанный риск, дополняет 3 линии §5.1/§5б.3):**
- 🤫 **Тихий** — длиннее, скрытнее, без тревоги, методичный перехват в уязвимой точке. Чистая работа.
- 🔥 **Рисковый** — быстрее, агрессивнее, ПОДНИМАЕТ ТРЕВОГУ → цель срывается в бег → подрезаешь на скорости красиво. Грязно, опасно, зрелищно.
**Тревога — инструмент, не наказание:** можно СПЕЦИАЛЬНО спугнуть цель, чтобы она побежала по предсказуемому эвак-маршруту, и снять в прыжке через атриум (манипуляция расписанием = высший пилотаж, как «гон NPC в точку» в Хитмане).

5т.6 **Упустить цель МОЖНО — фейл-стейт (даёт цене зубы, как смерть-зоны §5.3).** Провал = миссия фейл (рестарт мгновенный §23) ЛИБО грязная погоня-добивание (не убил тихо → охрана/побег → шумная эскалация: запасной геймплей, прощает ошибку, но грязнее и медленнее). Какой исход — следствие ВЫБОРА маршрута игрока.

5т.7 **Зрелищность = отдельная ось оценки (как ранг DMC / AC Contracts).** КАК убил: на пробеге / в прыжке / сквозь окно / подрезал бегущую / с высоты → стиль-очки. **PB-призрак теперь по 3 осям: время × фаза перехвата × стиль.** Спидранер гонит время, эстет — красоту, демон — всё. Режим разбора забега (§24.2-бис) показывает упущенное окно: «в след. раз перехвачу до того, как ушла в зал».

5т.8 **Структура — по МИССИЯМ:** каждая = здание/район + цель + её распорядок (clockwork) + N маршрутов (тихий/рисковый × линии §5б). Кампания = серия контрактов, рост сложности расписаний/охраны. Связь: §5 трассы → теперь миссии-здания; §5-бис якоря → теперь = подход-к-цели и точка убийства. Музыка (§3.4-бис): на момент килла бит проседает в «карман тишины» (как приглушение под голос), на выбросе — взрыв обратно.

5т.9 **ОТКРЫТО (дожать на болванке/после фила):** (а) расписание знаешь заранее (оптимизация под балконное окно, спид-ран-душа) vs узнаёшь по ходу (реактивный перехват) — склонились к Хитман-детерминизму (заучиваешь) + реактивный слой при тревоге; (б) баланс «тихий длинный vs громкий быстрый» — чтоб ни один не доминировал (логи §5б.6); (в) сколько боёвки «в ответ» нужно при погоне (минимум — мы атакующий призрак, не перестрелочник); (г) опознание цели в толпе на скорости — не сделать несправедливым.

5т.10 **ПРОКАЧКА — ВЕРТИКАЛЬНАЯ, общий герой на все миссии (решено 19.06.2026).** Рамиль выбрал чистую вертикальную прокачку (осознанно, после разбора конфликта с честностью). Персонаж РЕАЛЬНО качается (скорость/прыжок/живучесть/боевые), общий на всю кампанию контрактов — крючок и драйв грайнда. **Это ПИВОТ идентичности:** игра = RPG-экшен про ассасина с прокачкой; чистый глобальный спидран-лидерборд перестаёт быть святыней №1 (корректирует §17/§23/§9).
🚨 **Страховка от grind-to-win** (обязательна, иначе конкуренция гниёт в «кто дольше задрочил», нуб не догонит задрота, таблица мёртвая):
1. **Soft-cap.** Прокачка вертикальная, но упирается в ПОТОЛОК за вменяемое время (не сотни часов). Грайнд = путь наверх, не вечное преимущество. На эндгейме все сильны → решает скилл.
2. **Лиги по уровню прокачки + сезоны.** Соревнуешься с равными по прогрессу (ранги), сезонный сброс рейтинга = свежесть. Не кидаем нуба против макс-билда.
3. **На потолке вертикаль → билды (горизонталь, §9 ОЖИВАЕТ как эндгейм-мета):** достиг капа — дальше не «сильнее», а КАК распределил (скорость vs гаджеты vs стелс) под конкретный контракт. Билд + руки.
Итог: прогрессия полная (герой растёт через миссии), грайнд драйвовый, конкуренция живая (на потолке решают руки и билд, не часы).

**УТОЧНЕНО 19.06.2026 (Рамиль сам пришёл к решению честности — изящнее soft-cap):**
- **ДВА КЛАССА ЗАБЕГА = два лидерборда (его «голый vs фулл-прокач»):** 🔵 ГОЛЫЙ персонаж (база, все равны, ВЕЧНО честный спидран — хардкор-дисциплина à la low%) + 🟣 ПРОКАЧ (Open, RPG-песочница, билды+часы). Обе таблицы живые, не мешают. Это ключевая фишка удержания — две разные игры на одной миссии. (Поглощает идею Spec/Open §9.2 и снимает остроту grind-to-win — голый класс честен навсегда.)
- **ПЕРКИ — ДВА ТИПА (дизайн-приоритет на первый):**
  1. 🚪 **Возможности (маршрутные, ГЛАВНЫЕ):** пробитие стекла, пробитие стен, дольше вал-ран, новые входы. ОТКРЫВАЮТ маршруты, которых у голого НЕТ → прокач проходит там, где голый не может. Это не «быстрее по той же трассе», а ДРУГАЯ карта возможностей → разные дисциплины спидрана на одной миссии.
  2. ⚡ **Статы (числа):** скорость, высота прыжка, разгон. Та же линия быстрее. Вторичны.
- 🚨 **БАЛАНС-ПРИНЦИП (критично):** голый мувсет ПРОХОДИТ любую миссию сам — впритык, хардкорно, но проходимо. Перки = расширение/срезки/комфорт, НЕ костыль «без них никак». Иначе прокачка = обязаловка (grind-to-progress) и голый спидран как дисциплина умирает.

**ВРАГИ / УГРОЗЫ + КИБОРГ-ЗРЕНИЕ (решено 19.06.2026).** Ты КИБОРГ-воин → аугментация-зрение объясняет диегетически, почему видишь конусы/подсветку, и привязывает перки восприятия к лору. Слой угроз:
- 📹 КАМЕРЫ — статичные конусы, триггерят тревогу/снайперов. EMP-вырубить временно.
- 🎯 СНАЙПЕРЫ — держат зоны простреливания (подсвеченные линии огня), главная угроза на отходе/тревоге. Уходишь паркуром с линий.
- 🚁 ДРОНЫ — мобильные, патруль/преследование → ДИНАМИЧЕСКИЕ конусы (vs статичные камеры) → усложняют кайтинг (heist-процесс §кража/перехват).
- 🧍 ОХРАННИКИ-ЛЮДИ — патрули с конусами зрения. ПЕРК «как ветер» (прокачка): пронестись за спиной охранника / снять со спины на СКОРОСТИ так быстро, что не успел среагировать → буквальное воплощение «скорость=стелс» (§5т, не крадёшься а проносишься призраком).
- 👁️ КИБОРГ-ЗРЕНИЕ (аугментация, ПРОКАЧИВАЕМО): HUD подсвечивает конусы/линии огня/камеры. Качаешь восприятие → видишь камеры в БОЛЬШЕМ радиусе → планируешь маршрут заранее.
- ⚡ EMP-ГАДЖЕТ: вырубить ближайшую камеру на ~5 сек. ВРЕМЕННО и одноразово (на обратном пути уже не работает → нельзя зачистить всё, риск держится). Баланс-принцип: инструменты дают ОКНО, НЕ отменяют угрозу.
- 🚨 ГРАНИЦА (не уплыть в стелс): гаджеты работают НА ХОДУ, мгновенно (EMP на бегу, не «встань-примени-жди») — мы про скорость, флоу не рвём (связь §5т арсенал).

**ДЕТЕКТ (решено 19.06.2026):** тревога МГНОВЕННАЯ (засекли = цель сразу бежит), НО детект через ВИДИМЫЕ КОНУСЫ — камеры + vision-конусы охраны показаны → маршрут планируется заранее (обходишь конусы, как в классике стелса). Конусы детерминированы → заучиваемы (clockwork §5т.2, спидран-friendly). «Слепой» путь длиннее / больше паркура / выше скилл-требование.

**АРСЕНАЛ (решено 19.06.2026) — ПРАВИЛО: только летально-стильное, никакого огнестрела/нелетала (ломает концепцию «призрак-ассасин»).**
- Перки-двери (открывают убийство/проход): пробитие СТЕКЛА, пробитие тонких СТЕН (по умолчанию нельзя — открывается прокачкой).
- Крюк-кошка (прокачка: дистанция / скорость кошки) — вертикаль/манёвр.
- Тайминг-килл: на миг «стало невидно» → идеально прожать по таймеру = killshot (скилл-окно).
- Метательные: убийство с небольшой дистанции.
- НЕ берём: огнестрел, нелетальное — «хуйня, ломает концепцию».

**ПРОКАЧКА — список статов/возможностей (Рамиль 19.06.2026):** скорость бега, высота прыжка, длительность вал-рана, скорость НАБОРА скорости (разгон). + перки-двери выше. (что ещё — копим в банк.)
**МОДЕЛЬ ПРОКАЧКИ — РУЧНАЯ, опыт за успешную миссию (РЕШЕНО 19.06.2026, заменяет use-based/Скайрим).** Скайрим-авторост ОТМЕНЁН — не даёт ребилда и осознанного билда. Нужна РУЧНАЯ: «решил играть через кошку → вкачал кошку → жму срезки на кошке». Модель (стандарт Borderlands/PoE/Destiny, игрокам интуитивна):
- опыт капает ЗА ИГРУ (УСПЕШНЫЕ миссии — фарм вне миссии НЕ качает) → превращается в ОЧКИ (points);
- очки вкачиваешь ВРУЧНУЮ в перки/статы по выбору;
- + РЕСПЕК/РЕБИЛД → перестроил билд под стиль или конкретный контракт.
«Зарабатываешь играя, тратишь осознанно». Сходится с ограниченными архетипами билдов + видимым билдом соперника (§24.2-бис) → читаемая мета, ребилд под стиль. Привязка к аккаунту-персонажу. Перки-двери — за прогресс/очки.

**РЕДАКТОР / UGC = «КОНТРАКТ-РЕЖИМ» à la Hitman Contracts (решено 19.06.2026):** НЕ полный geometry-редактор (строить здания = перебор, годы работы, мусорный UGC). ОГРАНИЧЕННЫЙ редактор поверх ГОТОВЫХ выверенных карт: расставляешь ЦЕЛЬ(и) + тип контракта (§типы миссий) + условие (тихо/громко) + ПРОПЫ-шорткаты (бак запрыгнуть, усилить трафик, стройка, знаки → новые поверхности/срезки). Качество базовой карты гарантировано (игрок не ломает читаемость/проходимость). Модель: Trackmania/Mario Maker/Hitman Contracts. Всё сходится в ОДИН режим: редактор → Steam Workshop (§24.3) → другие бегают → лидерборд время/стиль. Бесконечный контент без строительства = сердце реиграбельности.
ГРАНИЦЫ (анти-мусор): (1) ставить только из ВЫВЕРЕННОГО набора пропов, не произвольную геометрию → сохраняет читаемость; (2) ВАЛИДАЦИЯ — автор обязан сам пройти контракт перед публикацией (Mario Maker-правило).

**ЛОР + ФРАКЦИИ (решено 19.06.2026) — контент-ось:** лор максимально простой — ты ФРИЛАНСЕР наёмный убийца, сам выбираешь у кого брать заказы → ФРАКЦИИ. Работаешь на фракцию → копишь её репу → открываются ЕЁ скины + ЕЁ цели. «Перекачивать фракции ради шмоток» = крючок удержания (Destiny/GTA/WoW-модель): вкачал одних → их шмот → вкачал других → их шмот. Одна механика = лор + прогрессия + скины + реиграбельность.
Усиления (19.06.2026): (1) фракция = свой СТИЛЬ контрактов (тихие/громкие/скоростные), не только скины → разнообразие миссий; (2) фракция = свой арсенал/гаджет (крюк/метательные/тайминг-килл, связь §5т); (3) РАЗВИЛКА конфликта (оценить): качать всех (макс удержание, меньше отыгрыша) vs взаимоисключающие конфликт-заказы (предал фракцию → цена репы, эксклюзив за лояльность, причина переигрывать). Склонность Рамиля — качать всех + капля конфликт-заказов для души.
**ТРИ ФРАКЦИИ (решено 19.06.2026) — каждая = эстетика + цели + мораль (= «типы жертв»):**
- 🔴 СОПРОТИВЛЕНИЕ (анти-корпорат, идея): цели — продажные корпораты/политики, кража корп.инфы, диверсии (взрыв заводов/офисов). Эстетика: панк/самоделки/партизан.
- 🟡 МАФИЯ/ЯКУДЗА (деньги/бизнес): цели — кто мешает бизнесу, должники, кража инфы на перепродажу, рэкет (взрыв машин/лавок как угроза). Эстетика: азиат-неон, кимоно, тату, золото.
- 🔵 КОРПОРАТЫ (власть/цинизм): цели — против бандитов/якудзы И против «хороших» (эколог-активист пропихивает невыгодный закон → «убрать»; лор объясняет, почему симпатичная цель мешает). Эстетика: глянец, хай-тек, дорого.
Сила: МОРАЛЬНАЯ СЕРОСТЬ = душа (валишь «хорошего» → иногда ты злодей; выбор фракции = моральный выбор «кто ты», обогащает кампанию, концовка зависит от фракции §кампания). Эстетика скинов по фракциям → 3 косметик-линейки, мотивация качать каждую (связь скины/монетизация). Три — ровно сколько надо.
**МУЗЫКА + ХАБ + ЭВОЛЮЦИЯ МИРА по фракциям (решено/мечта 19.06.2026) — рост показан МИРОМ, не цифрами:**
- 🎵 МУЗЫКА по доминирующей фракции (расширяет adaptive §3.4-бис): якудза → японское (сямисэн/Киото+неон), панк/сопротивление → гитары/панк, корпорат → лощёная классика/оркестр. Музыка реагирует не только на скорость, но и на «кто ты по духу». ЦЕНА: дёшево-средне (стемы по фракции). БЕРЁМ.
- 🏠 РАСТУЩАЯ КВАРТИРА-ХАБ = сердце: старт на ДНЕ (окно на мусорку) → с актами/прогрессом квартира ВЫШЕ → финал на вершине, из окна ВЕСЬ ГОРОД как карта. Интерьер по фракции (панк: гитары/граффити/рваный диван; якудза: азиат-лоск; корпорат: занос/стекло). Прямая визуализация кампании (подстава→бомж→наверх §кампания), эмоц.якорь (Hitman/GTA-хабы). ЦЕНА: средне (пресеты высоты+интерьера по актам), окупается эмоционально. БЕРЁМ.
- 🌃 ГОРОД МЕНЯЕТСЯ от влияния (больше якудзы → банзай/иероглифы на крышах; панки → граффити; корпораты → вылизанность) — САМ Рамиль: «супер-фантазия». ЦЕНА: AAA, дорого для соло (динамич. перестройка мира) → МЕЧТА/дальний прицел, не закладываемся. Дешёвый намёк: менять пару деталей в районе доминирующей фракции, не весь город.

**КАМПАНИЯ — лёгкая revenge-обёртка (решено 19.06.2026), клеит все системы:** сюжет = «был крутым убийцей → ПОДСТАВИЛИ → начинаешь с низов (как бомж) → работая на фракции копишь деньги → ПЛАТИШЬ ЗА ИНФОРМАЦИЮ → распутываешь, кто подставил». Оправдывает геймплей нарративно: подстава→обнуление = почему прокачка с нуля; инфа за деньги = почему гриндишь контракты. Подача через ПРОГРЕССИЮ, не катсцены (купил инфу → кусок правды + новая цель) — дёшево для инди, не ломает флоу. ФИНАЛ зависит от фракции → разные концовки = реиграбельность (связь с фракц.осью). Объём: лёгкая морковка ФОНОМ (записки/брифинги/радио между миссиями, не синематики), СКИПАБЕЛЬНА для чистого флоу. ⚠️ НЕ раздувать — тяжёлый сюжет в аркейде про скорость вредит. «Для галочки», как в Hitman, но даёт смысл гринду. ФИНАЛ (классика): последний контракт = убийство ТОГО, КТО ПОДСТАВИЛ. Вся кампания ведёт к нему → особый, самый зрелищный контракт (босс-уровень среди миссий).

**ТИПЫ КОНТРАКТОВ — способ устранения = тип миссии (решено 19.06.2026, разнообразие à la Hitman):**
- 🔪 Прямой килл — влетел/зарезал/ушёл (базовый).
- 💣 Саботаж/бомба — заложил на машину/объект → ушёл → взрыв. Новый ритм: ТАЙМЕР-эскейп (свалить до бабаха) + зрелищный финал.
- 🪤 Несчастный случай — подстроить (уронить груз с крана, столкнуть с высоты, подрезать). Чисто, без следов.
- 💼 Кража/перехват — НЕ убить, а украсть предмет/данные и сбежать (паркур-heist, цель = донести ноги). ПОДВАРИАНТ «heist с процессом» (Рамиль 19.06.2026): врываешься в офис → ставишь флешку (закачка-ТАЙМЕР) → охрана в режим ПОИСКА → кайтишь их паркуром по кругам, держишься живым пока качается → вернулся, забрал, ушёл. Обратный стелс (сам создал шум, танцуешь вокруг). Конусы охраны в search-паттерне (§детект) — читаешь и уходишь на скорости. Risk/reward по ДИСТАНЦИИ от точки: близко = качает быстрее но опаснее; далеко = безопасно но дольше под угрозой.
- 🎯 Множественная цель — несколько за забег, маршрут-головоломка.
- ⏱️ Окно времени — цель доступна N сек (связь фазы §5т.3).
Заказчик диктует УСЛОВИЕ: «тихо/без шума» vs «показательно/на публику» → меняет разрешённый стиль killа → разные стиль-очки. Один мувсет, разные задачи = реиграбельность.

**РЕЖИМЫ ИГРЫ (решено 19.06.2026):**
- **Дейлики** — классические: миссия за время / прыгни N / преодолей дистанцию → бонус денег+опыта.
- **ВИКЛИ-спецмиссия раз в неделю** — сильный retention/login-крючок (FOMO «зайду заберу»), даёт доп.валюту. Рамиль: на людях реально работает.
- **Бесконечка** — ОТКЛОНЕНО (не видится).
- **PvP-гонка** — идея: 2-3 игрока на миссии, кто первым добежал-замочил = полная награда. 🚨 Realtime-неткод на нашей скорости = риск §8 (Lemnis Gate закрылся на этом, для малой команды смерть). Дешёвая альтернатива с тем же ощущением: АСИНХРОННАЯ гонка против призраков (кто по времени первым к киллу). Вердикт: асинхрон сейчас, realtime — поздний/рискованный слой.
  - **СОПЕРНИКИ-АССАСИНЫ В МИССИЯХ (идея Рамиля 20.06.2026) — «оспариваемый контракт», PvP-ощущение БЕЗ неткода.** Обходит главный риск realtime-PvP (§выше): соперники = AI-боты (или async-призраки), НЕ realtime-игроки → сети нет, а «ты не один охотишься» есть. Стыковки: (1) ЛОР — фрилансер, контракт берут и другие наёмники (§390), город кишит убийцами → контракт оспаривается; (2) СТИМУЛ СКОРОСТИ — гонка за цель: убил первым = полная награда, опоздал = соперник забрал, тебе утешительное (давление воплощено в живом конкуренте, не абстрактный таймер); (3) ⭐ «УБРАТЬ СОПЕРНИКА, ЕСЛИ НЕ ЗАМЕТИЛ» = ПЕРЕИСПОЛЬЗОВАНИЕ механик: «скорость=стелс»/перк «как ветер» (§366) + конусы детекта (§371) на ПОДВИЖНОМ сопернике. Бежишь по крыше, видишь параллельного бегуна, он не засёк → снял со спины на скорости; засёк → он ускоряется/огрызается. Ноль новых систем, новая цель.
    - 🚨 ЧЕСТНОСТЬ: соперники = вариативность/помехи → зашумляют чистый спидран. РЕШЕНИЕ: режим с соперниками = ОТДЕЛЬНЫЙ режим + отдельная таблица (сегментация §428, как голый/прокач §356); чистый спидран остаётся чистым. Соперник детерминирован (clockwork §5т.2) + реактивный слой при обнаружении (§5т.3) → заучиваем маршрут, планируем перехват = и честно для спидрана, и убиваемо.
    - УРОВНИ РЕАЛИЗАЦИИ (дёшево→рискованно): (1) ⬅️ СТАВКА — AI-соперники (боты): без сети, полноценно реактивные/убиваемые, ядро фичи; (2) async-призраки реальных игроков как соперники: бегут рядом (из лидерборда §8), «фантомный тейкдаун» для красоты/бонуса, но на их записанное время не влияет (призрак = запись, не реагирует) — дёшево, «убийство» косметическое; (3) realtime-PvP — поздний/рискованный слой (§выше), не сейчас.
  - **Матчмейкинг по УРОВНЮ АККАУНТА, не по рейтингу/ELO (решено 19.06.2026):** подбор по уровню прокачки аккаунта (равные по прокачке). Внутри уровня решает СКИЛЛ. Награда: равный уровень → обычная; проигравший → утешительное (< победы, но НЕ ноль). 🚨 Анти-смурф через ПРЕМИУМ-модель (элегантно, без ELO-машинерии): фармить новичков можно только с нового нулевого аккаунта = НОВАЯ ПОКУПКА КОПИИ (дорого смурфить + лишняя продажа). А на нулевом аккаунте смурф = ГОЛЫЙ перс среди голых → тащит чистым скиллом, честно (его выбор, он заплатил). Стыкуется с голый-vs-прокач §5т.10: нулевой аккаунт = голая лига сама собой.

**ЗВАНИЯ/ТИТУЛЫ по lifetime-статистике (решено 19.06.2026):** агрегат накопленной статы (сколько пробежал / перелетел / убийств / S-рангов / часов вал-рана) → звания («Призрак 1-го класса» и т.п.). МНОГОУРОВНЕВЫЕ (тиры-лестница: новичок→…→легенда по каждой стате + общий ранг сверху) = длинная морковка, видимый прогресс. Почти бесплатно (стата уже пишется — логи §22 / реплеи §24). НЕ p2w (звание = титул, не сила → даже гринд-абуз безвреден). Флекс/идентичность/шеринг профиля → ретеншн. Слой профиля, не геймплея. **Реализация — STEAM из коробки** (Achievements за звания/вехи, статистика, витрина профиля, шеринг) → почти бесплатно, не свой бэкенд. Мелочь, поздний слой.

**ЭТАП 1 (приоритет, подтв. 19.06.2026):** минимальное движение + СУПЕР-минимальная миссия. Сезон/мета — отложено, не сейчас.

**СТРУКТУРА МИССИИ (3 акта, видение Рамиля 19.06.2026) — уточняет §5т.1:**
1. **Подход** — райончик, маркер цели, выбор пути (тихий-долгий / опасный-быстрый).
2. **Килл** — опасный путь → тревога → цель бежит → перехват в др. точке → стильный вход (крыша/стена/стекло) → kill.
3. **Отход** — наследил → тревога/снайперы держат зоны → симуляция погони, уход паркуром по линиям огня. Чисто сработал → спокойный уход (награда за чистоту). Снайперы/погоня = паркур-эвэйжн, НЕ перестрелка (мы атакующий призрак §5т.9в). Конец миссии → сезон/счёт.
⚠️ **Влияние на честность физики (§4.1/§23 детерминизм):** прокачка-статы делают забеги несравнимыми вне лиги своего уровня → глобальная таблица «по абсолютному времени» больше не работает как раньше. Лидерборды теперь СЕГМЕНТИРОВАНЫ (по лиге/капу). Детерминизм симуляции всё равно нужен (честность ВНУТРИ лиги, верифицируемые реплеи §24) — но «рекорды святы любой ценой» (§23) смягчается до «честно внутри своего сегмента».

## 24. РЕПЛЕИ, ШЕРИНГ И «ТЕАТР ПОВТОРОВ» (решено 18.06.2026)
База — transform/поза‑реплей (§21.3), лёгкий переносимый файл.
24.1 **Бэкенд лидербордов:** старт на **Steam Leaderboards + UGC‑призраки**, рекорд без смотрибельного реплея не засчитывается (честность через верификацию реплеев + модерация, как speedrun.com). Свой сервер — потом и только при упоре в то, чего Steam не даёт (глобальные синхронные daily, авто‑античит, веб‑таблицы). EOS не нужен (PC‑only).
24.2 **Проигрыватель реплея (scrubbing):** таймлайн‑бегунок (перемотка/пауза/замедление) + СВОБОДНАЯ камера (смотреть заезд изнутри движка с любого ракурса). Это сильнее видео — встроенный «театр повторов» для анализа/обучения/стримеров. Формат файла закладывать под scrubbing (таймстеп‑сэмплы + индекс по времени).
24.2-бис **РЕЖИМ РАЗБОРА ЗАБЕГА — «где проебался» (решено 18.06.2026).** Аналитический слой поверх скраба, почти бесплатный — данные уже пишутся (призрак = transform‑реплей). Состав:
- Скраб + замедление + покадровка + свободная камера (из 24.2).
- **График скорости на таймлайне** — видно визуально, где спидометр просел = где потерял темп.
- **Маркеры потери импульса** — точки на трассе, где сработал мягкий фейл (стена / промах wall‑tech §3.3‑тер). Подсвечено «здесь проебался».
- **Дельта к PB‑призраку** — наложить текущий заезд на лучший бок о бок: где отстал, где выиграл (Trackmania media tracker).
- **БИЛД соперника/призрака виден (решено 19.06.2026)** — смотришь не только КАК бежал, но и С ЧЕМ (распределение прокачки + перки): «вкачал вал‑ран + взял крюк → прошёл ту срезку». Обучение в комплекте: реплей + график скорости + БИЛД. Копируешь сборку + технику.
- **БИЛДЫ ОГРАНИЧЕННЫЕ, не бесконечные (решено 19.06.2026):** конечная билдовка → узнаваемые АРХЕТИПЫ (скоростной/боевой/стелс), читаемая мета, «посмотреть чужой билд» имеет смысл. Бесконечная кастомизация = нечитаемая каша, нельзя учиться/сравнивать. ⚠️ НЕ пережать в 1‑2 мета‑билда (все одинаковые, скучно) → цель: несколько ЖИЗНЕСПОСОБНЫХ архетипов, конечно но не узко.
Зачем: «проиграл и не понял почему» → «вижу, на спуске недокрутил». Ретеншн + самообучение. Scope: НЕ в первый плейбл (§1‑тер минимум), появляется со слоем реплеев/призраков — но технически бонус поверх уже записываемого призрака.
24.3 **Шеринг забегов комьюнити (виральность + ретеншн):**
- Прямой файл `.replay` (скинул другу → загрузил → бежит против твоего призрака).
- Steam Workshop (публикуй/подписывайся на заезды).
- Автоматом из лидерборда (топ‑N призраков подтягиваются на трассе через UGC).
24.4 **Формат файла реплея — заложить сразу:** id+версия трассы (валиден только для своей версии; иначе помечается несовместимым, не ломает забег); метаданные (ник/время/дата/сетап управления); квантование позы (лёгкость для мгновенного шеринга).
24.5 **speedrun.com (поздняя витрина):** их публичный REST API — читать таблицу/рекорды/ссылки на пруфы и показывать в игре. Писать туда автоматом нельзя (ручной сабмит с видео — их правила). Появляется, когда выйдем и комьюнити заведёт страницу. На старте — свой Steam‑лидерборд.
