00 О чём игра

В двух словах: ты гиперскоростной ассасин-киборг в неоновом мегаполисе. Берёшь контракты на заказные убийства и выполняешь их на полной скорости — несёшься по фасадам, влетаешь в окно, режешь цель не сбавляя темпа, вылетаешь в другое. Скорость = твоя невидимость: призрак, которого видят только мёртвым.
Триангуляция без прямого конкурента: раутинг Hitman × движение Mirror's Edge / Ghostrunner × зрелищность AC Contracts, в киберпанке.

Под боевиком — спидран-ядро: максимальная скорость, цена ошибки в долю секунды, накопление импульса за чистоту. Соревнование через призраков и сегментированные лидерборды. Контракты дают движению смысл, но НЕ заменяют его — сначала фил, потом содержание.

Главный принцип: 80% успеха = game feel. Лучшее — враг хорошего. Сначала фил на серой болванке, потом всё. Скорость заработана, не выдана. Главный враг — не рынок, а заброс на полпути.

Референсы

  • Ghostrunner — поток · Mirror's Edge — чистота · Hitman — clockwork-цели/раутинг
  • Trackmania — соревнование/дейлики · Neon White — скорость+лидерборды · Titanfall 2 — фил
  • Cyberpunk 2077 — город/неон/вертикаль · Stellar Blade — арт тела

L Лор и персонажи

Лёгкая revenge-обёртка, клеит все системы. Подаётся прогрессией (брифинги/радио/растущая квартира), НЕ катсценами. Скипабельна для чистого флоу. Не раздувать.

Протагонист — личность ≠ тело

Герой — киборг: постоянна ЛИЧНОСТЬ (голос, почерк, история), тело = сменный корпус. «Киборг» — внутренняя условность: объясняет, ПОЧЕМУ бегаешь по стенам и видишь конусы, но НЕ диктует вид.
  • Прошлое (решено): был независимым №1 города — фрилансер, к которому шли все три фракции, когда нужно «чисто». Прозвище за результат: после него ноль следов, ноль свидетелей.
  • Подстава = обнулили кибернетику. Аугментации выжгли / в чёрный список → очнулся в утильном корпусе на дне. Один поворот объясняет всю механику: прокачка = заново вживляешь железо; голый класс = базовое шасси; перки-двери = новые импланты.
  • Характер: немногословный, холодный, сухой чёрный юмор. Профессионал, у которого отняли всё и который спокойно идёт назад наверх по трупам.
  • Прозвище — открыто (рабочее НОЛЬ: обнуление + нейтральность к фракциям + призрак/ничто).

Арт облика = Stellar Blade (не хром CP2077)

Стилизованно-красивый человек (М/Ж) в облегающем техвире, импланты намёком (под кожей), не торчащее железо. Cyberpunk 2077 = только город/неон/вертикаль; тело = Stellar Blade.
  • Облик развязан с прокачкой — косметика 0 влияния на геймплей (не p2w по виду). «Обнулили железо» меняет способности, не внешку.
  • Каталог тел, НЕ редактор-лепка. Готовые авторские тела + модульные части + покраска + смена голов/причёсок. Слайдеры геометрии НЕ делаем (соло-дорого).
  • Сигнатурный неон-акцент (цвет/глиф «подписи») — читается на скорости при любом облике.

Антагонист — предатель-фиксер, который поднялся (решено)

Единственный, кому ты доверял — голос в наушнике годами, через кого шли контракты, кто держал твои данные. Не боец, а паук в сети (мозг/информация/контроль) — тематический контраст: ты тело и скорость, он информация.
  • Не просто продал — скоординировал сговор трёх фракций и занял твоё место, стал новым «королём» рынка убийств. Одно лицо = личное предательство + системный масштаб, без раздувания.
  • Мотив — амбиция: устал быть голосом в ухе чужой легенды → хотел быть королём, а не служить королям.
  • Финал кампании = последний контракт по нему. Концовки-по-фракции: контракт выдаёт та фракция, с кем ты в наибольшей репе; под чьим флагом город после.
  • ⚠️ Не путать: предатель-фиксер (антагонист) ≠ информатор (текущий союзник-источник). По нити информаторов выходишь на то, что координатор сговора = твой бывший человек.

Три фракции = эстетика + цели + мораль

Лор прост: ты фрилансер, сам выбираешь у кого брать заказы. Работаешь на фракцию → копишь репу → её скины + её цели + её музыка + её арсенал. «Перекачивать ради шмоток» = крючок удержания. Моральная серость — душа (валишь и «хороших»).
🔴 Сопротивление

Анти-корпорат. Цели: продажные корпораты/политики, диверсии. Панк/самоделки/партизан. Злая гитара.

🟡 Якудза

Деньги/бизнес. Цели: должники, кто мешает делам. Азиат-неон, золото, тату. Восточная музыка.

🔵 Корпораты

Власть/цинизм. Цели и «хорошие», что мешают. Глянец, хай-тек, дорого. Лощёный оркестр.

Прокачка — вертикальная, два класса забега

Персонаж реально растёт (общий на кампанию). Опыт за УСПЕШНЫЕ миссии → очки → вручную в перки (ручная модель Borderlands/PoE, есть респек/ребилд).
  • 🔵 Голый класс — база, все равны, вечно честный спидран (хардкор-дисциплина) · 🟣 Прокач (Open) — RPG-песочница, билды+часы. Два живых лидерборда = две игры на одной миссии.
  • Перки-двери (главные): пробитие стекла/стен, дольше вал-ран, крюк-кошка → ОТКРЫВАЮТ маршруты, которых у голого нет. Статы (скорость/прыжок/разгон) — вторичны.
  • ⚠️ Баланс-принцип: голый мувсет проходит любую миссию сам (впритык). Перки = расширение/срезки, НЕ костыль «без них никак».
  • Страховка от grind-to-win: soft-cap, лиги по уровню+сезоны, на потолке вертикаль → билды (горизонталь).

Первый плейбл — вертикальный срез для друзей

Минимум, чтобы фил читался, и ни байта больше. Цель — честный ответ «тянет ли перепройти»:
  • Одна трасса из gym-элементов: пропасть → вал-ран → мантл → спуск-слайд → финиш (серые блоки)
  • Спуск — кульминация в конце (забег кончается на пике ощущения)
  • Таймер + мгновенный рестарт (одна кнопка, без меню)
  • Свой призрак — со 2-го заезда, только лучший (PB) — один эталон, перезаписывается при побитии
  • Спидометр — виден разгон
НЕ входит: лидерборд, сеть, графика, 2-я трасса, контракты, звук кроме ветра.

03 Game feel — обязательное

Расширять окна по времени/позиции, чтобы игра подыгрывала игроку.

Прощающие окна

  • Coyote time (~5 кадров после края)
  • Jump buffering
  • Corner correction
  • Halved-gravity на пике прыжка

Камера продаёт скорость

  • FOV растёт со скоростью (главный трюк)
  • Наклон на вал-ране/поворотах
  • Motion blur на пике
  • Микро-тряска

Ядро — накопление скорости

  • Импульс не сбрасывается между движениями
  • Дольше без столкновений → быстрее (видимый индикатор)
  • Врезался = потерял разгон
  • Прыжок с платформы добавляет её импульс
  • Мувсет: бег, прыжок, рывок (КД), вал-ран, слайд, вскарабкивание/мантл

Закон высоты = скорости

Сохранение энергии, гравитация — друг на спуске:
  • Вниз → разгон — слайд по склону, вал-ран по нисходящей стене
  • Горизонт → держим — ровный бег/вал-ран сохраняет импульс
  • Вверх → платим — подъём и мантл тратят накопленное
Трасса = управление высотой: забрался дорого → слетел быстро. Высота = запасённая скорость (SSX, Tony Hawk, Dying Light).
⚠️ Разгон гравитацией FPS-зависим → лечим фикс-таймстепом (приоритет №1), иначе нечестные лидерборды.

Спасение по таймингу (wall-tech) — лестница под уровень игрока

Столкновение не провал, а 4 опции от safe к pro (парирование как в Sekiro, для геометрии):
  • 1. Объезд → не берёшь риск, медленнее но цел. Всем
  • 2. Блок (зажал) → теряешь часть, не всё. Без тайминга. Подушка новичку
  • 3. Парирование (попал в окно) → сохранил всё + буст/перенаправление. Мастер режет углы НАРОЧНО
  • 4. Лоб / ничего → полный сброс + микростан
Антиспам: окно парир. узкое. Порог входа: Trackmania-модель — темпом не давим, скорость ради таблицы. Один контакт = 4 прочтения под 4 уровня.

Обратная связь скорости — спидометр + сенсорика

Два разных слоя, нужны оба:
  • Спидометр = инфо (мозг) — точное число для оптимизации линии. Must-have для спидрана
  • Сенсорика = ощущение (тело) — FOV-кик + ветер (свист растёт, стрик-линии мимо камеры) + тряска + motion blur
⚠️ Только спидометр → втыкаешь в цифру, не в мир. Дополняет сенсорику, не заменяет. Приборка опциональна (трейлер прячет, оптимизатор разворачивает).

Gym-полигон для фила (этап 1 — не пустая коробка)

По одному серому эталонному элементу на каждую механику:
  • Пропасть → прыжок, рывок, coyote, buffer · Стена сбоку → вал-ран
  • Вертикальная стена/уступ → мантл · Горка-рампа → слайд+импульс · Серия платформ → чейнинг
Эти элементы — первые кирпичи модульной библиотеки трасс. Gym перерастает в стройматериал.

Музыка и звук

  • Жёсткая электроника — darksynth/DnB/industrial (школа Ghostrunner). Не эмбиент
  • Adaptive, не ритм-игра — слои нарастают с импульсом, проседают на ошибке. В бит попадать НЕ надо
  • 🚨 Клиентская, на детерминизм не влияет. 🚨 Болванку тестим БЕЗ музыки (маскирует кривой фил)

«Игра должна научить тебя быть быстрым» — Neon White. Скорость как награда: Tribes, Quake, Titanfall, Ultrakill.

05 Миссии — контракты + трассы

Часть 1 = закрытые миссии-уровни (не открытый мир). Катим контент частями, каждую доводим до кайфа, на закрытых отрабатываются основы. Лестница роста: трасса → район → город. Открытый город — видение части 2 (World Partition), сейчас не строим.

Дыхание миссии — 3 такта

  • Подход — паркур по крышам, копишь высоту=скорость, выбор пути (тихий-долгий / опасный-быстрый)
  • Килл — влетел в окно → цель живая, по расписанию (clockwork как Hitman) → перехват → стильный вход → kill. Был быстр — взял на балконе; затупил — цель ушла, перестраиваешь дугу на лету
  • Отход — наследил → тревога/снайперы держат зоны → уход паркуром (НЕ перестрелка, мы атакующий призрак)
Здание прошивается насквозь: влетел-резанул-вылетел = 2-3 сек. Трекер цели (силуэт сквозь стены) — честность, опознаёшь ещё с крыши.

✅ Скилл-срез — святое

Через механики: накопил импульс, тормознул ради разгона, безумный прыжок. Не трогаем. Чекпоинты = якоря (где побывать), не рельсы (как собрать).

❌ Чит-срез — убираем

Эксплойт коллизии, физ-баг, телепорт сквозь геометрию.

Угрозы + детект (киборг-зрение подсвечивает)

  • Камеры (статичные конусы), снайперы (линии огня), дроны (динамичные конусы), охрана (патрули). EMP на ходу = окно, не отмена угрозы
  • Детект через ВИДИМЫЕ конусы, детерминированы → заучиваемы (clockwork, спидран-friendly). Тревога мгновенная
  • Соперники-ассасины (AI-боты, отдельный режим+таблица): оспариваемый контракт, тейкдаун соперника через те же конусы

Смерть и фейл

Мягкий (лоб/промах wall-tech → потеря импульса), жёсткий (пропасть/смерть-зона → мгновенный рестарт). Упустить цель = фейл ИЛИ грязная погоня. Смерть дешёвая, цикл «ещё разок» священен.

Типы контрактов: прямой килл · саботаж/бомба (таймер-эскейп) · несчастный случай · кража/перехват (паркур-heist) · множественная цель · окно времени. Заказчик диктует «тихо/громко».

23 Платформа и честность

PC-only, Steam

  • Консоли — нет (спидран-сообщество на ПК, сертификация не нужна)
  • Идентичность = RPG-ассасин с контрактами+прокачкой; спидран-слой внутри
  • Детерминизм — приоритет №1. Лидерборды СЕГМЕНТИРОВАНЫ по лиге/капу (честно внутри сегмента), верифицируемые реплеи

Управление

  • Клава+мышь — основа. Весь дизайн, баланс, спидран-мета под неё + ребинды
  • Геймпад — поддерживается, но не затачиваем

Обязательные спидран-фичи (в архитектуру сразу)

  • Мгновенный рестарт (1 кнопка, 0 загрузки) · встроенный таймер + splits
  • Призрак своего PB в реальном времени · practice mode (рестарт с сегмента)
  • Верифицируемые смотрибельные реплеи рекордов
Steam даёт почти бесплатно: лидерборды, облако, воркшоп.

21.9 Конфликты инструментов

Стек — не просто список: инструменты дерутся за одно и то же. ⭐ = решать на болванке ПЕРВЫМ.

⭐ 1. PBCharacterMovement ↔ GASP — кто владеет движением

Оба претендуют на управление персонажем: PBCMC = физика/импульс, GASP = свой Character с Motion-Matching-traversal (мантл/climb на своём движении). Нельзя взять оба вслепую.
Решение: физику владеет PBCMC, GASP низводится до анимслоя (AnimBP + Motion Matching читают скорость/состояние, играют позы), его управление движением отключаем. ⚠️ Цена: traversal GASP завязан на его CMC → мантл/climb переносим руками. Проверить ПЕРВЫМ — развилка всего фила.

⭐ 2. Детерминизм ↔ CMC — ловушка

Включить Substepping/Async Physics НЕ делает CMC детерминированным: Substepping для Chaos-симуляции, а CMC двигает капсулу кинематически на game-tick → остаётся FPS-зависимым. Нужен свой фикс-аккумулятор вокруг тика движения, либо миграция на Mover 2.0 (async, детерминирован by design) после нахождения фила. Не самообман «включил Substepping = честно».
3. Motion Matching ↔ детерминизм

Анимация = клиентский визуал, на тайминги не влияет — ЕСЛИ движение не root-motion-driven. Правило: капсулу двигает физика, анимация поверх.

4. MetaHuman-скелет ↔ анимации

Разные скелеты → нужен ретаргет (IK Rig + Retargeter). На болванке — голый Mannequin, MetaHuman = арт-слой потом.

5. Polyart ↔ MetaHuman

Две ветки персонажа. Основная = MetaHuman (Stellar-Blade-вайб); Polyart = прототип рантайм-кастомизации. Не держать обе как основные.

6. Ghost Replay ↔ AutoReplay

Transform vs input. Ghost Replay основной (устойчив к недетерминизму), AutoReplay только backup. Не смешивать форматы.

7. SteamSAL ↔ лишние Steam-плагины

AdvancedSessions/EnhancedOnlineSessions тянут OSS Steam и конфликтуют при init. Нужны только лидерборды → ставим ТОЛЬКО SteamSAL.

8. City Sample ↔ минимализм/FPS

Спроектирован под Nanite/Lumen, этап 1 их выключает. Трение на арт-слое. Whitebox на Quaternius.

21.7 Лицензии — правило выживания

🚨 ГЛАВНОЕ ПРАВИЛО: перед любым GitHub/Fab-ассетом В КОД — проверять лицензию. GPL/AGPL/LGPL/CC-BY-SA = ЯД (копилефт открывает исходники всей игры или нельзя линковать с UE по EULA Epic). «No license» = по умолчанию копирайт = НЕЛЬЗЯ.

✅ Чисто для платного Steam

  • MIT/Apache/BSD — линкуется с UE, продажа OK
  • CC0 — общественное достояние, без атрибуции (Polyhaven, ambientCG, Kenney, Quaternius)
  • Fab Standard — Personal=Professional права, коммерция OK; нельзя лишь перепродавать ассет
  • UE-Only — коммерция OK, но ТОЛЬКО в UE-проекте (City Sample, GASP)
  • RF royalty-free — Mixamo, Sonniss (⚠️ запрет AI/ML-тренинга)

⛔ Чёрный список

  • ExtendedCharacterMovement GPL-3.0 — мина, выкинуто (§04)
  • DaftMover ? — лицензия не подтверждена (в старом плане ошибочно «MIT»). PBCharacterMovement замещает
  • Villeek/MovementDemo ? — готовый шаблон гонки, но лицензии нет → только референс
  • asee5g/Character-Customization no license
  • itch-UI-киты — часто только PNG + AI-генерёжка, копирайт мутный
Роялти UE: 5% только с дохода свыше $1M lifetime (Epic Games Store — royalty-free). На старте неактуально, в финмодель при успехе.
📦 Склад ≠ проект: бесплатную/дедлайн-халяву Fab забирать в библиотеку свободно (навсегда, 0 затрат). Дисциплина «whitebox первым» — только про ИМПОРТ в проект. Раз в 2 недели чистить Fab Limited-Time Free.
