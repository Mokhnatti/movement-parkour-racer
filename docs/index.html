00 О чём игра

В двух словах: несёшься через неоновый киберпанк-город на бешеной скорости — бежишь по стенам, прыгаешь между крышами, скользишь под препятствиями. Чем чище проходишь — тем быстрее летишь. Один забег = несколько секунд чистого потока и адреналина. А потом сравниваешь время с друзьями и всем миром: рядом бежит призрак чемпиона, и ты пытаешься его обогнать.
Жанр одной строкой: скоростной паркур-рейсер от 3 лица — будто Ghostrunner встретил Trackmania. Игра про скорость, точность и желание пройти трассу ещё чище.

Максимальная скорость, цена ошибки в долю секунды, накопление импульса за чистоту прохождения. Соревнование через призраков и лидерборды на сложных трассах — авторских и генерируемых.

Главный принцип: 80% успеха = game feel. Лучшее — враг хорошего. Сначала фил на серой болванке, потом всё. Скорость заработана, не выдана. Главный враг — не рынок, а заброс на полпути.

Референсы

  • Ghostrunner — поток
  • Warframe — импульс
  • Trackmania — соревнование/дейлики
  • Neon White — скорость + лидерборды
  • Titanfall 2 — фил

Первый плейбл — вертикальный срез для друзей

Минимум, чтобы фил читался, и ни байта больше. Цель — честный ответ «тянет ли перепройти»:
  • Одна трасса из gym-элементов: пропасть → вал-ран → мантл → спуск-слайд → финиш (серые блоки)
  • Спуск — кульминация в конце (забег кончается на пике ощущения)
  • Таймер + мгновенный рестарт (одна кнопка, без меню)
  • Свой призрак — со 2-го заезда, только лучший (PB) — один эталон, перезаписывается при побитии
  • Спидометр — виден разгон
НЕ входит: лидерборд, сеть, графика, 2-я трасса, звук кроме ветра. Вся петля «беги → ошибись/спасись → финишируй быстрее → ещё разок» в одном куске.

03 Game feel — обязательное

Расширять окна по времени/позиции, чтобы игра подыгрывала игроку.

Прощающие окна

  • Coyote time (~5 кадров после края)
  • Jump buffering
  • Corner correction
  • Halved-gravity на пике прыжка

Камера продаёт скорость

  • FOV растёт со скоростью (главный трюк)
  • Наклон на вал-ране/поворотах
  • Motion blur на пике
  • Микро-тряска

Ядро — накопление скорости

  • Импульс не сбрасывается между движениями
  • Дольше без столкновений → быстрее (видимый индикатор)
  • Врезался = потерял разгон
  • Прыжок с платформы добавляет её импульс
  • Мувсет: бег, прыжок, рывок (КД), вал-ран, слайд, вскарабкивание/мантл

Закон высоты = скорости

Сохранение энергии, гравитация — друг на спуске:
  • Вниз → разгон — слайд по склону, вал-ран по нисходящей стене
  • Горизонт → держим — ровный бег/вал-ран сохраняет импульс
  • Вверх → платим — подъём и мантл тратят накопленное
Трасса = управление высотой: забрался дорого → слетел быстро. Высота = запасённая скорость (SSX, Tony Hawk, Dying Light).
⚠️ Разгон гравитацией FPS-зависим → лечим фикс-таймстепом (приоритет №1), иначе нечестные лидерборды.

Спасение по таймингу (wall-tech) — лестница под уровень игрока

Столкновение не провал, а 4 опции от safe к pro (парирование как в Sekiro, для геометрии; рефы Titanfall, Mirror's Edge):
  • 1. Объезд → не берёшь риск, медленнее но цел. Всем
  • 2. Блок (зажал) → теряешь часть, не всё. Без тайминга. Подушка новичку
  • 3. Парирование (попал в окно) → сохранил всё + буст/перенаправление. Мастер режет углы НАРОЧНО
  • 4. Лоб / ничего → полный сброс + микростан
Антиспам: окно парир. узкое — зажал заранее = только блок. Честность → фикс-таймстеп. Порог входа: Trackmania-модель — темпом не давим, скорость ради таблицы, не ради «пройти». Один контакт = 4 прочтения под 4 уровня. Отскок вверх → высота → скорость.

Обратная связь скорости — спидометр + сенсорика

Два разных слоя, нужны оба:
  • Спидометр = инфо (мозг) — точное число для оптимизации линии. Must-have для спидрана (Trackmania, bhop-ups)
  • Сенсорика = ощущение (тело) — FOV-кик + ветер (свист растёт со скоростью, стрик-линии/частицы мимо камеры, искажение краёв) + тряска + motion blur
⚠️ Только спидометр → втыкаешь в цифру, не в мир. Дополняет сенсорику, не заменяет.
Приборка спидранера (спидометр + импульс/чистота + таймер + сплиты) — опциональна: трейлер/новичок прячут (чистый кадр), оптимизатор разворачивает. Бонус: цифра растёт на спуске / падает на подъёме — обучает невидимым законам.

Gym-полигон для фила (этап 1 — не пустая коробка)

По одному серому эталонному элементу на каждую механику, иначе нечего тюнить:
  • Пропасть → прыжок, рывок, coyote time, buffer
  • Стена сбоку → вал-ран
  • Вертикальная стена / уступ → вскарабкивание / мантл
  • Горка-рампа → слайд + импульс вниз
  • Серия платформ → чейнинг прыжков, сохранение импульса
Эти элементы — первые кирпичи модульной библиотеки трасс. Gym перерастает в стройматериал, не выкидывается.

Рывок по КД = тактический ресурс

Ценен не сам рывок, а тайминг траты:
  • Тайминг важнее дистанции — путь длиннее выгоден, если ставит рывок в идеальную точку
  • Комбо: рывок в слайд / с горки >> рывок «в пустоту» → линия строится вокруг связок
  • Растратил не вовремя → ждёшь КД, теряешь темп
Развилки риск/награда рождаются сами, из КД + геометрии. Игрок считает тайминги.

Музыка и звук

  • Тон: жёсткая электроника — darksynth / DnB / industrial (школа Ghostrunner: Daniel Deluxe, Dan Terminus). Не эмбиент
  • Adaptive, не ритм-игра — музыка реагирует на игрока: слои нарастают с импульсом, расцветают на чистом разгоне, проседают на ошибке. Попадать в бит НЕ надо (убило бы свободу линий)
  • Музыка = второй индикатор импульса ушами — расцвела = в потоке, оборвалась = облажался
  • 🚨 Клиентская, на детерминизм не влияет. 🚨 Болванку тестим БЕЗ музыки (маскирует кривой фил)
  • Продакшн: Рамиль спросит знакомого музыканта. Не бюджетный блокер

«Игра должна научить тебя быть быстрым» — Neon White. Скорость как награда: Tribes, Quake, Titanfall, Ultrakill.

05 Трассы — чекпоинты-якоря + свобода

Часть 1 = закрытые трассы-уровни (не открытый мир). Почему: катим контент частями · каждую карту доводим персонально до кайфа · на закрытых трассах отрабатываются основы — все великие гонки начинали так. Цель: пара трасс, по которым реально интересно бегать.
Лестница роста: трасса → район → город. Накопились трассы в одном сеттинге → сшиваем в район (мини-открытый мир, старые трассы = маршруты внутри + связки). Районы стыкуются → вырастает город. Контент катится частями, риск растёт плавно, модули §6 переиспользуются.
Открытый город — видение части 2 (если доживём): трасса = A→Б по якорям, поиск маршрута на игроках (модель Mirror's Edge Catalyst). Технически — World Partition (UE 5.8) + та же модульная библиотека. Сейчас не строим: масштаб мира убил бы отработку основ.

Площадка для спидранов → emergent-маршруты это ФИЧА, не баг.

Смерть и фейл-стейты

Без смерти у риска нет цены. Два типа фейла:
  • Мягкий — лоб в стену / промах wall-tech → потеря импульса + микростан, бежишь дальше
  • Жёсткий — пропасть / смертельная зона (обрыв, лазеры) → мгновенный рестарт
Смерть дешёвая и мгновенная (без загрузок — цикл «ещё разок» священен). Смерть-зоны — атрибут РИСКОВЫХ линий: безопасная линия без пропастей (медленно, но не убивает), срез над бездной = ставка «влетел или сдох».

✅ Скилл-срез — святое

Через механики: накопил импульс, тормознул ради разгона, безумный прыжок, хитрый маршрут. Не трогаем. Душа спидрана.

❌ Чит-срез — убираем

Эксплойт коллизии, физ-баг, телепорт сквозь геометрию.
Чекпоинты = якоря, не рельсы: «здесь надо реально побывать на скорости», но НЕ «как собрать». Между якорями — полная свобода маршрута. 3 линии (безопасная/быстрая/безумная) — подсказки-ориентиры. Мастер найдёт 4-ю — это победа дизайна.

Философия Quake/Titanfall/Ghostrunner: bunny hop и rocket jump родились багами → стали ядром жанров. Патчить только то, что ломает физику насквозь.

Режим разбора забега — «где проебался»

Аналитика поверх скраба, почти бесплатна (данные призрака уже пишутся):
  • Замедление + покадровка + свободная камера
  • График скорости на таймлайне — видно, где спидометр просел = где потерял темп
  • Маркеры потери импульса — точки, где влетел в стену / промазал wall-tech. Подсвечено «здесь проебался»
  • Дельта к PB-призраку — наложить заезд на лучший бок о бок: где отстал, где выиграл
«Проиграл и не понял почему» → «вижу, на спуске недокрутил». Не в первый плейбл — приходит со слоем реплеев.

12-13 Тех-стек и Claude/MCP

  • UE 5.8, НЕ UE6 — UE6 сырой/ломающий до 2027+. Большие карты решены в 5.8 (World Partition).
  • Фиксированный таймстеп физики — заложить сразу (честные лидерборды).
  • World Partition — авто-стриминг, длина трассы почти не влияет на FPS. Lumen/Nanite на болванке выкл (300+ FPS).
MCP-поправка: официальный плагин UE 5.8 НЕ умеет PIE/скриншоты/консоль из коробки (только спавн актёров/материалы/автотесты). Для «Claude играет → смотрит → крутит фил» нужен сторонний сервер (StraySpark с PIE / UnrealClaude с захватом вьюпорта) или свой toolset. allowlist обязателен, auto-approve ВЫКЛ.
