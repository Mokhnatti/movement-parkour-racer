## ЧТО ЗА ИГРА (одной строкой)
Скоростной паркур-рейсер от 3 лица в киберпанке + слой RPG-ассасина на контрактах + спидран. «Скорость = стелс» (призрак, которого видят только мёртвым). UE 5.8, PC/Steam, премиум-модель.

## ТВЁРДЫЕ РЕШЕНИЯ (не пересматривать без причины)
- Камера 3-е лицо. Движок UE 5.8 (НЕ UE6). Фикс-таймстеп физики (честность лидербордов). Соревнование — асинхрон (призраки+лидерборды), НЕ realtime-неткод. Параметры мувмента наружу (DataAsset/BP).
- 80% успеха = game feel: фил на серой болванке в тишине ПЕРВЫМ, потом всё. Скорость заработана, не выдана.

## КЛЮЧЕВЫЕ ДИЗАЙН-РЕШЕНИЯ (детали в PLAN.md)
- **Движение:** закон высота=скорость (§3.3-бис); рывок по КД = тактический ресурс; wall-tech лестница объезд/блок/парир/лоб (§3.3-тер); мувсет бег/прыжок/рывок/вал-ран/слайд/мантл.
- **Пивот в RPG-ассасина (19.06):** контракты (§5-тер), вертикальная прокачка + общий герой (§5т.10), ДВА класса забега = два лидерборда (🔵 голый честный / 🟣 прокач Open).
- **Три фракции** (Сопротивление/Якудза/Корпораты) = эстетика+цели+мораль; репа → скины/арсенал/музыка.
- **Кампания-месть:** подстава → с низов → инфа за деньги → финал убить подставившего.
- **Соперники-ассасины в миссиях (20.06):** PvP-ощущение без неткода (AI-боты), оспариваемый контракт, тейкдаун соперника через конусы детекта (§417).
- **Протагонист (20.06, §14.2-тер):** личность ≠ тело; «киборг» = внутренняя условность (объясняет способности), НЕ диктует вид; облик развязан с прокачкой. Арт = STELLAR BLADE (красивый человек М/Ж + импланты намёком, НЕ хром-протезы). Подстава = обнулили кибернетику → прокачка-с-нуля на лоре. Облик через КАТАЛОГ тел + модульные части + покраска, БЕЗ слайдеров-лепки.
- **Монетизация:** premium/buy-to-play + косметика (гринд + донат-престиж), НЕ p2w.
