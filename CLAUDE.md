## ПИВОТ ИДЕНТИЧНОСТИ (решено 19.06) → RPG-ЭКШЕН ПРО АССАСИНА
Игра = контракты заказных убийств + спидран. «Скорость = стелс» (призрак, которого видят только мёртвым). Спидран — сильный слой, но честность теперь ВНУТРИ сегмента (лиги/капа), лидерборды сегментированы. Детали → PLAN §5-тер, §5т.*, §23.
- **Вертикальная прокачка**, общий герой; ДВА класса забега = два лидерборда (🔵 голый честный / 🟣 прокач Open). PLAN §5т.10.
- **Три фракции** (Сопротивление/Якудза/Корпораты) = эстетика+цели+мораль; репа → скины/арсенал/музыка. PLAN §фракции.
- **Кампания-месть**: подстава → с низов → инфа за деньги → финал = убить подставившего. PLAN §кампания.
- **Соперники-ассасины в миссиях** (PvP-ощущение без неткода, AI-боты): оспариваемый контракт, тейкдаун соперника через конусы детекта. PLAN §417.

## ПРОТАГОНИСТ + КАСТОМИЗАЦИЯ (решено 20.06) → PLAN §14.2-тер
- **Личность ≠ тело.** Постоянна личность (голос/почерк/история/прозвище — ОТКРЫТО), тело = сменный корпус.
- **«Киборг» = внутренняя условность** (объясняет способности), НЕ диктует вид. Облик развязан с прокачкой (не p2w по виду).
- **Арт облика = STELLAR BLADE** (красивый стилизованный человек М/Ж + импланты намёком, НЕ хром-протезы CP2077; CP2077 = только город/неон).
- **Подстава = обнулили кибернетику** → прокачка-с-нуля и голый/прокач — на лоре.
- **Облик через КАТАЛОГ тел + модульные части + покраска, БЕЗ слайдеров-лепки.** Цвет/материал/swap частей = дёшево; лепка геометрии = НЕ делаем.
